## What is Open Community Groups?

Open Community Groups (OCG) is an open source platform that helps communities run groups, events, and speaker programs in one place. It provides dedicated public sites and dashboards for communities, groups, and individual members.

<Note>
  OCG is currently in beta and under active development. Features may change between releases.
</Note>

## How OCG is organized

OCG has one public site and three role-based dashboards:

<CardGroup cols={2}>
  <Card title="Public site" icon="globe" href="/guides/public-site">
    Where anyone can discover groups and events, browse in Explore, RSVP to events, check in on the day, and submit to Call for Speakers.
  </Card>
  <Card title="User dashboard" icon="user" href="/guides/user-dashboard">
    Manage your personal activity — upcoming events, profile, invitations, session proposals, and submissions.
  </Card>
  <Card title="Community dashboard" icon="building" href="/guides/community-dashboard">
    Community-wide setup and management: identity, team access, regions, categories, analytics, and groups.
  </Card>
  <Card title="Group dashboard" icon="users" href="/guides/group-dashboard">
    Where organizers run groups and events — manage events, members, sponsors, and communications.
  </Card>
</CardGroup>

## Key features

<CardGroup cols={3}>
  <Card title="Multi-community support" icon="sitemap">
    Multiple communities can share one installation, each with dedicated sites and dashboards.
  </Card>
  <Card title="Rich event management" icon="calendar">
    Host in-person, virtual, or hybrid events with multi-session agendas, speaker management, and QR check-in.
  </Card>
  <Card title="Call for Speakers" icon="microphone">
    Open a CFS, receive proposals, review and accept sessions, and notify speakers automatically.
  </Card>
  <Card title="Zoom integration" icon="video">
    Automatic Zoom meeting creation for virtual and hybrid events and sessions.
  </Card>
  <Card title="Interactive discovery" icon="map">
    Browse events and groups with map and calendar views, filtered by region, category, and type.
  </Card>
  <Card title="Automated notifications" icon="bell">
    Email notifications for registration, reminders, updates, and custom messages with calendar attachments.
  </Card>
</CardGroup>
